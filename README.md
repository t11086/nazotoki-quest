# ナゾトキクエスト 〜知識で解く事件簿〜

小6〜中3向けの謎解きアドベンチャー型知育アプリ。
「勉強した知識が、事件を解くカギになる」がコンセプト。

- 企画の詳細: [docs/企画書.md](docs/企画書.md)
- ステータス: Phase 1 プロトタイプ実装ずみ(事件1話+デイリー捜査)

## 概要

プレイヤーは新米探偵として、学園や街で起きる事件を捜査する。
暗号解読(英語)、アリバイ計算(数学)、現場の痕跡分析(理科)、古地図の読解(社会)など、
教科の知識を「証拠」として使いながら1話10〜15分の事件を解決していく。

## 実装ずみの機能(Phase 1)

- 🗂️ **事件簿 FILE 1『消えた文化祭の看板』** — 手がかり収集 → 教科クイズ(英語・数学・理科)→ 最終推理 → 解決編
- 🗓️ **デイリー捜査** — 日替わりの1問+連続捜査日数(ストリーク)
- 🏅 **探偵ランク** — 推理ポイントで 見習い探偵 → 探偵助手 → 一人前探偵 → 名探偵
- 💡 3段階ヒント(使うと獲得ポイント減)、誤答ペナルティ
- 📱 スマホファーストUI、進行データは localStorage 保存(アカウント不要)

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # 型チェック+本番ビルド(dist/)
npm run preview  # ビルド結果をローカル配信
```

技術構成: React 18 + TypeScript + Vite(バックエンドなし)
